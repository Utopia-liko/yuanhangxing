# 远航星播放器

## 项目说明

本项目基于 [GuanyuPlayer](https://github.com/PizzaDark/GuanyuPlayer) 项目进行二次修改开发

- **原作者**：[PizzaDark](https://space.bilibili.com/6297797)
- **二次开发**：Liko(https://space.bilibili.com/507900516)

## 功能介绍

- 按下快捷键 Alt+Y+H 播放/停止音频
- 语音中检测到关键词自动播放（支持多关键词，需要语音模型）
- 支持自定义快捷键、音量调节、开机启动等
- 支持多音乐文件随机播放（默认模式/自定义模式）
- 可自定义语音识别模型路径

## 更新日志

v1.0功能：

- **多关键词支持**：支持最多10个关键词，任意匹配即可触发
- **默认模式**：使用 yuanhangxing.mp3（单曲）
- **自定义音乐管理**：支持添加最多10个自定义音频文件，随机播放
- **自定义模型路径**：模型已内置到程序中，也可自由选择语音识别模型位置，不再限制固定目录
- **音乐试听功能**：在音乐管理器中可预览试听音频文件
- **配置文件管理**：可直接打开配置文件所在位置
- **界面优化**：新增关键词编辑器、音乐管理器等独立窗口

## 文件说明

远航星播放器.exe         - 主程序
yuanhangxing.mp3            - 默认模式音频文件（必需）
vosk-model-small-cn-0.22/  - 语音模型（可选）

## 快速开始

1. 双击运行`build.bat`
2. 或者手动输入：
3. 在`dist`目录下找到`.exe`并运行

## 语音识别（可选）

1. 下载模型：<https://alphacephei.com/vosk/models>
   推荐选择：vosk-model-small-cn-0.22（也可用其他语言的版本）
2. 解压到任意位置
3. 在程序设置中点击"浏览"按钮，选择模型文件夹
   （也可以解压到程序目录，程序会自动检测）
4. 运行程序，麦克风检测到任意关键词即可自动触发播放

## 快捷键设置

默认快捷键：Alt + Y + H
修改方法：点击"修改快捷键"按钮，按下新组合键
恢复默认：点击"恢复默认"按钮

## 关键词设置

默认关键词：远航星（最多10个）
修改方法：点击"编辑关键词"按钮，每行输入一个关键词
恢复默认：点击"恢复默认"按钮
说明：语音识别时会检测任何一个关键词，匹配到则自动触发播放

## 音乐设置

**默认模式：**

- 默认模式 - 使用 yuanhangxing.mp3（单曲）

**自定义模式：**

- 点击"管理音乐"按钮
- 最多添加10个音乐文件（支持mp3/wav/ogg格式）
- 选中文件后点击"试听"可预览
- 触发播放时将随机播放列表中的一首

**说明：**

- 删除仅从列表移除，不会删除磁盘文件
- 可随时切换回默认模式
- 配合mixline等软件或虚拟声卡可用于游戏语音连麦

## 系统托盘

- 关闭窗口 → 最小化到托盘
- 双击托盘图标 → 显示窗口
- 右键托盘图标 → 菜单选项
- 选择"退出" → 完全退出程序

## 常见问题

Q: 快捷键没反应？
A: 检查是否有其他软件占用，或修改为其他组合

Q: 语音识别不工作？
A: 1. 检查语音模型是否正确放置或路径是否正确
2\. 检查麦克风是否正常工作
3\. 确认语音识别选项已启用
4\. 检查是否选择了正确的音频输入设备

Q: 如何切换音乐播放模式？
A: 在音乐管理器中可以选择默认模式或自定义模式

## 系统要求

- Windows 7/8/10/11 (64位)
- 内存 2GB 以上
- 麦克风（语音识别需要）

## 配置文件

位置：C:\Users<用户名>.yuanhangxing\_config.json

**说明：**

- 存储快捷键、音量、关键词、音乐列表等所有设置
- 删除此文件可重置所有设置为默认值
- 可在程序中点击"打开配置文件位置"按钮快速定位

**配置项包括：**

- hotkey: 快捷键组合
- volume: 音量（0.0-1.0）
- keywords: 关键词列表
- audio\_files: 音乐文件列表
- model\_path: 自定义模型路径
- auto\_start: 开机启动设置
